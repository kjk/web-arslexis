Noah Pro can use a wealth of different dictionaries (and it's possible to create your own dictionaries).

There are two main types of dictionaries:

  • Devil's Dictionary, a satirical dictionary by Ambrose Bierce
  • dictionaries based on WordNet database.
In order to accommodate needs of different users Noah Pro provides many versions of WordNet database. Each database contains a different set of words and takes a different amount of memory. In short:
  • mini (wn_mini.pdb), small (wn_small.pdb), medium (wn_medium.pdb) and full (wn_full.pdb) are databases of increasing size. Each one contains more words than its predecessor but also requires more memory.
  • a definition in Noah consists of explanations and sample phrases. Some users prefer to save memory and not to have sample phrases. For those users each database comes in lite version without sample phrases (wn_mini_lite.pdb, wn_small_lite.pdb, wn_medium_lite.pdb, wn_full_lite.pdb, wn_advanced_lite.pdb). Lite version requires roughly 30% less memory.
  • some users don't need definitions of simple words. They should find advanced (wn_advanced.pdb) version of the database useful.
We recommend using the biggest database you can afford (in terms of memory taken on your PalmPilot). Here's a detailed information on every available databases:

Please note: to minimize the bandwidth consumption only the following dictionaries are included with the main distribution file: wn_small.pdb, wn_medium.pdb, wn_advanced.pdb and wn_full.pdb. You can download the rest for free.
