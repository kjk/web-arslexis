Introduction and system requirements.

Noah Pro 1.0 for Windows is an English dictionary for Microsoft Windows operating system. It has definitions of 122 thousand words.

  Noah Pro 1.0 requires Windows 95, Windows 98, Windows ME, Windows NT 4.0, Windows 2000, Windows XP or newer.

Download and install

After you purchase Noah Pro, you'll receive an e-mail with instructions on how to download Noah Pro. The file you'll download is NoahProSetup.exe. Please run this file to install Noah Pro.

How to use

Noah Pro is extremely easy to use. Here's how it looks at startup:

 

  To obtain a definition of the word enter it in the entry field or scroll the list of word to the desired word.

Description of menus

To exit Noah Pro select "Exit" from "Program" menu
You can change the font used for the word list using "Choose font" menu.

  Word definition consist of main definition text, examples and synonyms. You can switch displaying examples and synonyms off by using "Hide synonyms" and "Hide examples" menus.

Menu "About" will display basic information about Noah.
Menu "Manual..." will display Noah manual in a new browser window.
Menu "Check version..." will check if you have the latest version of Noah
Menu "Visit website..." will take you to ArsLexis website.
Last three option require internet connection.

  Enjoy using Noah!
