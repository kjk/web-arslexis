Noah Lite 1.0 for Windows overview

Noah Lite 1.0 for Windows is a free English dictionary for Microsoft Windows operating system. It has definitions of only 36 thousand words (compared to Noah Pro's 122 thousand) but other than that it works the same as Noah Pro.

System requirements

Noah Lite 1.0 requires Windows 95, Windows 98, Windows ME, Windows NT 4.0, Windows 2000, Windows XP or newer.

How to download it

Noah Lite 1.0 is free and you can download it here.
