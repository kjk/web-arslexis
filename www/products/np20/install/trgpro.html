Home » Noah Pro manual : Installing Noah Pro 2.0 on TRGPro

Short instructions

Noah Pro can read database that resides on a Compact Flash extension memory card of TRGPro. In that mode Noah Pro requires 1 MB of internal memory and 5 MB on external memory to run.

Note: method explained below is the only safe way to have Noah's database on TRGPro. In particular you should not just put databases on CF card and use AutoCF as it might crash Noah.

To install Noah Pro make sure that you (temporarily) have 5200 K of free internal memory on your TRGPro (read how to do it).

Install the following files on your device:

  • noah_pro.prc (the reader program)
  • db_ex\wn_ex_full.pdb (database file compatible with CF card). You must install this and only this database.
After installation use CFPro application on your device to move "Wordnet ex full.pdb" to CF card. You must move the database under root ("\") directory (otherwise Noah won't find it).

Noah should find the database on the external memory card upon startup.

Detailed instructions

If the above instruction is not sufficient, here are detailed installation steps.

Launch Palm Desktop application and press "Install" button on the left ( )

On the windows that appears, press "Add..." button, then navigate to the folder where you unpacked Noah. Select "noah_pro" and press "Open" button in the file dialog.

Press "Add" button again, navigate to "db_ex" director, select "wn_full_ex.pdb" and press "Open" button.

Press "Done" button:

You'll see the final confirmation, press "OK" button and do a Hot Sync.

Installation steps on the device

In CFPro application switch to CF card. Make sure that you have at least 5.1 M of free space. Navigate to "\" folder as shown on the left. Note: you must use "\" folder, otherwise Noah won't find the database.

Switch to internal memory view, find "Wordnet ex full" file, tap on it to highlight and press "Copy" button.

Press "OK" on the confirmation dialog. CFPro will move the database from internal memory to CF card.
Switch again to extension memory card view to verify that the database has been moved. You can now delete it from the internal memory using "Delete" button.

Noah should now find and use the database on MemPlug external memory card.
