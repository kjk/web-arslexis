Home » Noah Pro manual : Installing Noah Pro 2.0 on a HandEra

Short instructions

Noah Pro can read database that resides on a HandEra's SD card or Compact Flash card. If installed on SD or CF external memory card Noah Pro requires 1 MB of internal memory and 5 MB of external memory to run.

To install Noah Pro make sure that you (temporarily) have 5200 K of free internal memory on your PDA (read how to do it).

Install the following files on your device:

  • noah_pro.prc (the reader program)
  • db_ex\wn_ex_full.pdb (database file compatible with external memory card). You must install this and only this database.
After installation use CardPro application on your device to move "Wordnet ex full.pdb" to external memory card. You must move the database under ""\PALM\LAUNCHER" directory (otherwise Noah won't find it).

Noah should find the database on the external memory card upon startup.

Detailed instructions

If the above instruction is not sufficient, here are detailed installation steps.

Launch Palm Desktop application and press "Install" button on the left ( )

On the windows that appears, press "Add..." button, then navigate to the folder where you unpacked Noah. Select "noah_pro" and press "Open" button in the file dialog.

Press "Add" button again, navigate to "db_ex" director, select "wn_full_ex.pdb" and press "Open" button.

Press "Done" button:

You'll see the final confirmation, press "OK" button and do a Hot Sync.

Installation steps on the device

Using CardPro application to an extension memory card (either CF or SD). Make sure that you have at least 5.1 M of free space. Navigate to "\PALM\LAUNCHER" folder as shown on the left. Note: you must use "\PALM\LAUNCHER" folder, otherwise Noah won't find the database.

Switch to internal memory view, find "Wordnet ex full" file, tap on it to mark and press "Copy" button.

CardPro will copy the database from internal to external memory.
Switch again to extension memory card view to verify that the database has been copied. You can delete the database from internal memory.

Noah should now find and use the database on HandEra external memory card.
