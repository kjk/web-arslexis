Home » Noah Pro manual : Installing Noah Pro 2.0 on a MemPlug

Short instructions

Noah Pro can read database that resides on a MemPlug, SmartMedia card adaptor for Handspring Visor. If installed on MemPlug external memory card Noah Pro requires 1 MB of internal memory and 5 MB of external memory to run.

To install Noah Pro make sure that you (temporarily) have 5200 K of free internal memory on your Visor (read how to do it).

Install the following files on your device:

  • noah_pro.prc (the reader program)
  • db_ex\wn_ex_full.pdb (database file compatible with MemPlug). You must install this and only this database.
After installation use PiMover application on your device to move "Wordnet ex full.pdb" to SmartMedia memory card. You must move the database under "\PALM\" directory (otherwise Noah won't find it).

Noah should find the database on the external memory card upon startup.

Detailed instructions

If the above instruction is not sufficient, here are detailed installation steps.

Launch Palm Desktop application and press "Install" button on the left ( )

On the windows that appears, press "Add..." button, then navigate to the folder where you unpacked Noah. Select "noah_pro" and press "Open" button in the file dialog.

Press "Add" button again, navigate to "db_ex" director, select "wn_full_ex.pdb" and press "Open" button.

Press "Done" button:

You'll see the final confirmation, press "OK" button and do a Hot Sync.

Installation steps on the device

In PiMover application switch to extension memory card. Make sure that you have at least 5.1 M of free space. Navigate to "\PALM\" folder as shown on the left. Note: you must use "\PALM\" folder, otherwise Noah won't find the database.

Switch to internal memory view, find "Wordnet ex full" file, tap on it to mark and press "Move..." button. PiMover will move the database from internal to external memory.

Switch again to extension memory card view to verify that the database has been moved.

Noah should now find and use the database on MemPlug external memory card.
