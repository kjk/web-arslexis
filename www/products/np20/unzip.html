Home » Noah Pro 2.0 manual : How to unzip Noah Pro 2.0

Noah Pro 2.0 comes in noah_pro2.zip archive file that needs to be uncompressed. There are many programs to do it, e.g. WinZip for Windows, ZipIt or Stuffit Expander for Macintosh.

You can read more about the process here (see point 3).

Important note for Macintosh users: it is crucial that you have a correct settings for your unzipping program.

In Zip It the option in preferences is called by default, smart-strip linefeeds from text files and should be turned off.

In Stuffit Expander change program settings to never convert text files.

Different settings may corrupt the database file which might make Noah crash.
