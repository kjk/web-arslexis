Noah Pro 2.0 overview

Noah Pro 2.0 is an English dictionary for PalmPilot-compatible PDAs (Palm, Visor, Sony CLIE, HandEra, TRGPro, IBM WorkPad). Noah Pro costs $19.95 and you can purchase it here.

Noah offers definitions of 122 thousand words (is based on a WordNet lexical database).

Thanks to excellent compression all 122 thousand words only take 4.7 MB of memory. It also has smaller databases that require less memory, so that you can choose the one that best fit your needs.

Noah Pro 2.0 also supports many external memory cards:

  • Palm m105/m125/m500/m505 SD/MMC memory card
  • Sony CLIE MemoryStick
  • MemPlug for Visor
  • HandEra 330 and its SD/CF external memory slots
  • TRGPro and its CF external memory slot
If database is stored in external memory Noah requires 1 MB of internal memory and 5 MB of memory on an external memory card.

With Noah Pro you can also create your own dictionaries.

System requirements

Noah Pro requires a PalmOS device with OS 3.0 or greater (all currently sold devices and most of older ones). Supported models include: Palm III, IIIx, IIIxe, IIIc, Vx, VIIx, m100, m105, m125, m500, m505, all Handspring Visor models, all Sony CLIE models, HandEra 330, TRGPro and others.

Noah Pro requires from 1M (the smallest dictionary with 13 thousand words) up to 4.7 M (the biggest dictionary with 122 thousand words). We do not recommend running Noah Pro on handhelds with less than 4 MB total internal memory.

If a database is stored on an external memory card, Noah Pro requires 1 MB of internal memory and 5 MB of external memory.
