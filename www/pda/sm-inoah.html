iNoah

iNoah 1.0 (download free trail version inoah_sm_1_0.cab, 76 kB) is a wireless English dictionary. iNoah is a true dictionary on-the-go. It offers definitions of 140 thousand words, uses only 76 kB of memory, color display of definitions and ability to adjust definition font size.

iNoah requires MS Smartphone with wireless internet data connection. This application is not signed and might not work on application locked phones. Downloading definitions uses data connection.
