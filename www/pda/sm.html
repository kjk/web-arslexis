ArsLexis offers the following software for MS Smartphones: iPedia, iNoah.

iPedia

iPedia 1.0 (download free trial version ipedia_sm_1_0.cab, 60 kB) is a wireless encyclopedia. iPedia offers more than 440 thousand articles on all possible subjects. It uses very little memory, offers quick lookup of encyclopedia articles thanks to optimized network protocol, display of articles is optimized for small screen and user interface targeted at Smartphone devices.

It requires MS Smartphone with wireless internet data connection. This application is not signed and might not work on locked phones. Downloading articles uses data connection.

iNoah

iNoah 1.0 (download free trail version inoah_sm_1_0.cab, 76 kB) is a wireless English dictionary. iNoah is a true dictionary on-the-go. It offers definitions of 140 thousand words, uses only 76 kB of memory, color display of definitions and ability to adjust definition font size.

iNoah requires MS Smartphone with wireless internet data connection. This application is not signed and might not work on application locked phones. Downloading definitions uses data connection.

ArsLexis also offers software for Pocket PC and Palm.
