Home » Documentation » iNoah for MS Smartphone overview

iNoah is an English dictionary for MS Smartphone, optimized for mobile, wireless devices.

Requirements: iPedia requires an MS Smartphone (e.g. MPx200, i600, SPV E100 and E200 etc.) with wireless internet connection. It's not signed so it might not work on application locked devices.

Please try the FREE TRIAL version first to check compatibility.

Why is iNoah better than any other dictionary? It provides definitions of 140 thousand words but is using only 75 kB of memory (other dictionaries offering similar number of words use around 5 MB of memory, 80 times as much).

iNoah features:

  • definitions of 140 thousand words
  • quick lookup of word definitions thanks to optimized network protocol
  • display of definitions optimized for small screen
  • customizable font size (3 different sizes) for comfortable viewing
  • list of recently looked up words
  • user interface optimized for MS Smartphone devices
