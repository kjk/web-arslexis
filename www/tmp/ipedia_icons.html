Overview

ArsLexis needs an icon for an upcoming Palm OS application.

The application is called iPedia and it's a wireless encyclopedia based on Wikipedia. Wireless means that it only works on Palm OS devices like Treo that have wireless internet connection.

Possible methaphores for the application (and therefore an icon) are things related to knowledge (like books) maybe with the "wireless" accents. But that's just an (obvious) idea - we do welcome (expect, actually) better ideas than that. We don't want themes similar to Wikipedia's logo as we're not affiliated with them.

Icons for Palm OS have slightly different requirements than icons for Windows or Mac OS. They need to be delivered as *.bmp files. We need them in following formats:

  • a small icon in standard (15x9) and high (30x18) resolution, both in black&white (2 colors) and 256 color versions (i.e. 4 versions in total). Small icon is less important than the regular icon.
  • regular icon in standard (22x22) and high (44x44) resolution, also 2 version: black&white (2 colors) and 256 color version.
  • color version must use a very specific color palette. You can extract the palette from the example icons below.
  • one of the colors must be designated as the transparent, background color. It can be any color that is not used for the icon graphic itself. In the example below it's the yellowish color
