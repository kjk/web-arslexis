Q. I get "Database not found" error on startup

It means that only the reader program has been installed (noah_lite.prc). Please also install a database (wordnet_medium.pdb) on your device as explained in manual. The database must be installed in internal memory. Noah Lite doesn't support external memory card such as MMC/SD on Palm, MemoryStick on Sony CLIE, MemPlug, HandEra/TrgPro CF card etc. Only Noah Pro supports reading the database directly from the external memory card. You can read more about Noah Pro here.

Q. I use Macintosh and I got garbage on the screen or program crashes.

It is very important to have correct settings for unzipping program for unpacking noah_lite.zip. Some unzippers have settings to convert text files to Mac text files. This alters content of files and corrupts them.

In ZipIt the preference "by default, smart-strip linefeeds from text files" should be turned off.

In Stuffit Expander change settings to "never convert text files".

Q. Does Noah Lite support external memory cards (e.g. MMC/SD card on Palm, Memory Stick CLIE etc.) ?

Noah Lite doesn't support external memory cards. Only Noah Pro 2.0 supports the following external memory cards:

  • Palm SD/MMC external memory card (present e.g. in Palm m105, m125, m500 and m505)
  • Sony CLIE MemoryStick card
  • MemPlug external memory card for Visor
  • HandEra 330 Compact Flash card or SD card
  • TRGPro Compact Flash card
If a database is put in external memory Noah requires 1 MB of internal memory and 5 MB of external memory to run.

Q. The database is too big.

Noah Pro has better compression (databases take less memory) as well as more choice of databases.

Q. Can I make my own dictionary database?

Only Noah Pro allows you to create your own databases.

Q. Does Noah Lite support Springboard memory extension for Visor?

Yes, you can put Noah Lite and on Visor's 8 MB flash module (but not the backup module). Just move Noah Pro and a databases of your choice to flash module using File Mover application.

Q. What are advantages of Noah Pro?

Here's a detailed explanation. In short:

  • Better compression - saves up to 450 kB of memory for other programs
  • Support for many external memory cards:
    • Palm m105/m125/m500/m505 SD/MMC memory card
    • Sony CLIE MemoryStick
    • MemPlug for Visor
    • HandEra 330
    • TRGPro
  • Many user interface enhancements:
    • ability to copy definitions to a clipboard
    • support for on-screen keyboard
    • history of 5 last accessed words
    • ability to scroll either by tapping or using hardware buttons
    • ability to scroll by line, half a page or a page
  • Much more databases (7 more, 11 in total)
  • Ability to create your own dictionaries
