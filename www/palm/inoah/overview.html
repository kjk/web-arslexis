iNoah is an English dictionary for Palm, optimized for mobile, wireless Palm devices like Treo 600/650.

Requirements: Palm OS device with wireless internet connection (e.g. Treo 600).

Please try the FREE TRIAL version first to check compatibility.

Why is iNoah better than any other dictionary? It provides definitions of 140 thousand words but is using only 63 kB of memory (other dictionaries offering similar number of words use around 5 MB of memory, 80 times as much).

iNoah is built on technology created for Noah Pro and provides the following features:

  • definitions of 140 thousand words
  • quick lookup of word definitions thanks to optimized network protocol
  • display of definitions optimized for small screen
  • copy & paste definition to clipboard for use in other programs
  • bookmark interesting words for future reference
  • list of recently looked up words
  • 3 different ways of displaying the definition
  • display customization - choose your own fonts and colors
  • support for latest Palm OS 5 devices
  • support for 5-Way Navigator in Treo, Palm Tungsten
