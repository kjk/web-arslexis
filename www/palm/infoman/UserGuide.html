Thank you for trying a trial version of InfoMan, a wireless information agent for Palm devices, optimized for Treo 600/650.

InfoMan requires a Palm device with wireless internet connection (e.g. Treo smartphone).

To use InfoMan please install InfoMan.prc on your device. Trial version has limits number of times you can use InfoMan. To remove the limit please purchase registration code and register InfoMan using menu 'Options/Register'.
