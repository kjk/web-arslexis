Home » Documentation » InfoMan for Palm overview

InfoMan is a wireless information agent for Palm OS, optimized for Palm smartphones like Treo.

Requirements: Palm OS device with wireless internet connection (e.g. Treo 600/650).

Please try the FREE TRIAL version first to check compatibility.

InfoMan provides the following features:

  • weather information
  • U.S. movie times
  • free U.S. box office information
  • currency conversion
  • stock quotes and portfolio management
  • U.S. 411 and phone book (white pages, yellow pages) queries
    • find a phone number for a person/business (white-pages and yellow-pages information)
    • find person information given a phone number
    • find zip codes for a given city
    • find a city given a zip code
    • find area codes for a given city
    • find a city given area code
    • find international calling number for a given country
  • cheapest gas prices in your area
  • free jokes
  • support for latest Palm OS devices
  • support for 5-Way Navigator in Treo, Tungsten
