Noah Lite
FREE! Noah Lite 1.0 is a free version of Noah. It's a great product, sharing many features of Noah Pro, such as:
  • efficient use of memory thanks to high compression of data
  • clean and simple user interface
  • support for latest Palm OS 5 devices
  • support for 5-Way Navigator in Treo, Palm Tungsten

Main differences between Noah Pro and Noah Lite are:

  • Noah Pro contains more definitions (122 thousand vs. 37 thousand)
  • Noah Lite doesn't support external memory cards
You can: download Noah Lite, read the manual.
